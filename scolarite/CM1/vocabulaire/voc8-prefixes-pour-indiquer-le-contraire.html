La leçon

Le préfixe du contraire

Pour former le contraire d'un mot, on peut employer des préfixes.

Un préfixe se place devant le radical (le mot de base) pour former un nouveau mot de sens opposé.

Rappel : un préfixe ne change pas la classe grammaticale du mot. Un adjectif reste un adjectif, un verbe reste un verbe, etc.
🔴

Les préfixes in- / im- / il- / ir-

in-

Le préfixe in- s'utilise devant la plupart des lettres.

inconnu – invisible – incapable – injuste
im-

Devant les lettres m, b et p, le préfixe in- devient im-.

impossible – immobile – imbattable – imprudent
Astuce : devant m, b, p → on met im- (pas de « n » devant ces lettres !).
il-

Devant la lettre l, le préfixe in- devient il-.

illisible – illogique – illégal – illimité
ir-

Devant la lettre r, le préfixe in- devient ir-.

irréel – irrégulier – irresponsable – irréparable
🔶

Les préfixes dé- / dés-

dé- / dés-

Le préfixe dé- s'utilise devant une consonne.

Le préfixe dés- s'utilise devant une voyelle.

défaire – démonter – décoller – débrancher
désobéir – désagréable – désordre – désarmer
🔸

Le préfixe mal- / malheur-

mal- / malheur-

Le préfixe mal- indique quelque chose de négatif ou fait mal.

malhonnête – malheureux – maladroit – malchance – malpropre
✎️

Exercice 1 – Trouve le contraire avec un préfixe

Ajoute le bon préfixe (in-, im-, il-, ir-) pour former le contraire.

1 possible →
2 connu →
3 lisible →
4 réel →
5 patient →
6 légal →
7 mobile →
8 régulier →
✅ Voir la correction ▼
1) impossible (devant p → im-)
2) inconnu (in-)
3) illisible (devant l → il-)
4) irréel (devant r → ir-)
5) impatient (devant p → im-)
6) illégal (devant l → il-)
7) immobile (devant m → im-)
8) irrégulier (devant r → ir-)
📖

Exercice 2 – Préfixe dé- ou dés- ?

Ajoute le préfixe dé- ou dés- pour former le contraire.

1 faire →
2 obéir →
3 monter →
4 agréable →
5 coller →
6 ordre →
7 brancher →
✅ Voir la correction ▼
1) défaire (dé- devant consonne)
2) désobéir (dés- devant voyelle)
3) démonter (dé- devant consonne)
4) désagréable (dés- devant voyelle)
5) décoller (dé- devant consonne)
6) désordre (dés- devant voyelle)
7) débrancher (dé- devant consonne)
🤔

Exercice 3 – Quel préfixe ?

Trouve le préfixe utilisé et recopie le mot de base (radical).

1 malhonnête → préfixe ___ + mot de base ___
2 irresponsable → préfixe ___ + mot de base ___
3 désarmé → préfixe ___ + mot de base ___
4 illimité → préfixe ___ + mot de base ___
5 imbattable → préfixe ___ + mot de base ___
6 maladroit → préfixe ___ + mot de base ___
✅ Voir la correction ▼
1) préfixe mal- + mot de base honnête
2) préfixe ir- + mot de base responsable
3) préfixe dés- + mot de base armé
4) préfixe il- + mot de base limité
5) préfixe im- + mot de base battable
6) préfixe mal- + mot de base adroit
💭

Exercice 4 – Complète les phrases

Écris le contraire du mot en gras en utilisant un préfixe.

1 Ce texte est lisible. → Ce texte est ___.
2 Cette règle est juste. → Cette règle est ___.
3 Il faut obéir aux règles. → Il ne faut pas ___.
4 Ce garçon est adroit. → Ce garçon est ___.
5 Maman va monter l'armoire. → Maman va ___ l'armoire.
6 Ce film est réel. → Ce film est ___.
7 Cet exercice est possible. → Cet exercice est ___.
✅ Voir la correction ▼
1) Ce texte est illisible (il- devant l)
2) Cette règle est injuste (in-)
3) Il ne faut pas désobéir (dés- devant voyelle)
4) Ce garçon est maladroit (mal-)
5) Maman va démonter l'armoire (dé- devant consonne)
6) Ce film est irréel (ir- devant r)
7) Cet exercice est impossible (im- devant p)
⭐

À retenir

Pour former le contraire d'un mot, on peut ajouter un préfixe :

in- (devant la plupart des lettres) : inconnu, injuste

im- (devant m, b, p) : impossible, immobile

il- (devant l) : illisible, illégal

ir- (devant r) : irréel, irrégulier

dé- (devant consonne) / dés- (devant voyelle) : défaire, désordre

mal- : malheureux, maladroit

Mots clés
in- im- il- ir- dé- / dés- mal-
