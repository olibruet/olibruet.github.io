La leçon

Règle générale

Pour former le féminin d'un nom, on ajoute en général un -e à la fin du mot.

🔈 un ami → une amie • un employé → une employée
Les terminaisons spéciales
Terminaison masc. Terminaison fém. Exemples
-teur → -trice un spectateur → une spectatrice
-eur → -euse un chantreur → une chanteuse
-er → -ère un prisonnier → une prisonnière
-ien → -ienne un musicien → une musicienne
-on / -an → -onne / -anne un paysan → une paysanne
-prince → -esse un prince → une princesse
Noms totalement différents

Certains noms ont une forme féminine complètement différente :

🔈 un roi → une reine • un homme → une femme • un père → une mère
un garçon → une fille • un frère → une sœur • un neveu → une nièce
Astuce : Quand tu hésites, pose-toi la question : quelle est la terminaison du nom masculin ? Choisis la règle qui correspond.
✎️

Exercice 1 – Forme le féminin

Donne le féminin de chaque nom masculin.

1 un acteur →
2 un magicien →
3 un menteur →
4 un prince →
5 un roi →
6 un manipulateur →
7 un prisonnier →
8 un chanteur →
✅ Voir la correction ▼
1) une actrice (-teur → -trice)
2) une magicienne (-ien → -ienne)
3) une menteuse (-eur → -euse)
4) une princesse (-ce → -cesse)
5) une reine (form différente)
6) une manipulatrice (-teur → -trice)
7) une prisonnière (-er → -ère)
8) une chanteuse (-eur → -euse)
🔎

Exercice 2 – Trouve la règle utilisée

Pour chaque paire, indique quelle règle a été appliquée (+ e, -trice, -euse, -ère, -ienne, forme différente).

1 un employé → une employée
2 un ouvreur → une ouvreuse
3 un spectateur → une spectatrice
4 un homme → une femme
5 un musicien → une musicienne
6 un boulanger → une boulangère
✅ Voir la correction ▼
1) + e (règle générale)
2) -eur → -euse
3) -teur → -trice
4) forme différente
5) -ien → -ienne (doublement de consonne)
6) -er → -ère
💬

Exercice 3 – Complète les phrases

Complète avec le féminin du nom entre parenthèses.

1 La ___ de la Nuit chante sur scène. (roi)
2 Cette ___ joue du violon dans l'orchestre. (musicien)
3 La ___ est enfermée dans le palais. (prisonnier)
4 La ___ en scène dirige le spectacle. (metteur)
5 Paméla est la ___ de Sarastro. (prince)
6 Une ___ applaudit depuis sa place. (spectateur)
✅ Voir la correction ▼
1) La Reine de la Nuit chante sur scène.
2) Cette musicienne joue du violon dans l'orchestre.
3) La prisonnière est enfermée dans le palais.
4) La metteuse en scène dirige le spectacle.
5) Paméla est la princesse de Sarastro.
6) Une spectatrice applaudit depuis sa place.
