O18 – Les adverbes en -ment

Orthographe • Leçon 18

CM1 • Dictée Le Haka (Nouvelle-Zélande)

La leçon Mots de la dictée Exercice 1 Exercice 2
📚

À retenir

Règle générale

La plupart des adverbes qui se terminent par -ment se forment à partir de l'adjectif au féminin suivi du suffixe -ment. Le e peut devenir un é.

fort → forte → fortement
simple → simple → simplement
brutal → brutale → brutalement
-ent → -emment

Les adjectifs qui se terminent par -ent forment leurs adverbes en -emment.

violent → violemment
récent → récemment
-ant → -amment

Les adjectifs qui se terminent par -ant forment leurs adverbes en -amment.

méchant → méchamment
bruyant → bruyamment
Astuce : pour savoir s'il faut écrire -emment ou -amment, retrouve l'adjectif !
• Si l'adjectif finit par -ent → adverbe en -emment (violent → violemment)
• Si l'adjectif finit par -ant → adverbe en -amment (méchant → méchamment)
Attention : -emment et -amment se prononcent pareil !

Exercice 1 – Forme l'adverbe

Transforme chaque adjectif en adverbe en -ment.

1. simple → simplement
2. brutal → brutalement
3. violent → violemment
4. méchant → méchamment
5. fort → fortement
6. général → généralement
7. récent → récemment
8. bruyant → bruyamment
9. ancien → anciennement
10. amical → amicalement
✅ Voir la correction ▼
1. simple → simplement (féminin simple + ment)
2. brutal → brutalement (féminin brutale + ment)
3. violent → violemment (-ent → -emment)
4. méchant → méchamment (-ant → -amment)
5. fort → fortement (féminin forte + ment)
6. général → généralement (féminin générale + ment)
7. récent → récemment (-ent → -emment)
8. bruyant → bruyamment (-ant → -amment)
9. ancien → anciennement (féminin ancienne + ment)
10. amical → amicalement (féminin amicale + ment)
📝

Exercice 2 – Complète avec l'adverbe

Complète chaque phrase avec l'adverbe formé à partir de l'adjectif entre parenthèses.

1. Les danseurs chantent le haka violemment. (violent)
2. Ils regardent méchamment leurs adversaires. (méchant)
3. Les guerriers tirent fortement la langue. (fort)
4. Le haka est généralement pratiqué avant un match. (général)
5. Ils se tapent brutalement la poitrine et les cuisses. (brutal)
6. La démonstration se déroule simplement. (simple)
7. Le public applaudit bruyamment. (bruyant)
8. Cette danse a été récemment pratiquée lors d'une cérémonie. (récent)
9. Les équipes se saluent amicalement après le match. (amical)
10. Cette danse était anciennement réservée aux guerriers. (ancien)
11. Les joueurs frappent bruyamment leurs cuisses. (bruyant)
12. Cette tradition témoigne particulièrement du respect des ancêtres. (particulier)
✅ Voir la correction ▼
1. violemment (violent → -ent → -emment)
2. méchamment (méchant → -ant → -amment)
3. fortement (fort → forte + ment)
4. généralement (général → générale + ment)
5. brutalement (brutal → brutale + ment)
6. simplement (simple → simple + ment)
7. bruyamment (bruyant → -ant → -amment)
8. récemment (récent → -ent → -emment)
9. amicalement (amical → amicale + ment)
10. anciennement (ancien → ancienne + ment)
11. bruyamment (bruyant → -ant → -amment)
12. particulièrement (particulier → particulière + ment)
