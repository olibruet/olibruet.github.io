Qu'est-ce qu'un multiple ?

Multiple

On appelle multiple un nombre qui peut s'écrire sous la forme d'un produit de deux nombres entiers.

42 est un multiple de 6 puisque 42 = 6 × 7
42 est un multiple de 7 puisque 42 = 7 × 6
🔎

Qu'est-ce qu'un diviseur ?

Diviseur

On dit que 6 et 7 sont des diviseurs de 42.

42 a d'autres diviseurs : 1, 2, 3, 6, 7, 14, 21 et 42.

42 = 1 × 42
42 = 2 × 21
42 = 3 × 14
42 = 6 × 7
Astuce : pour trouver tous les diviseurs d'un nombre, cherche toutes les multiplications qui donnent ce nombre.
💡

À savoir : reconnaître les multiples

Règles à retenir
  • • Les multiples de 2 sont tous des nombres pairs. Ils se terminent par 0, 2, 4, 6 ou 8.
  • • Les multiples de 5 se terminent toujours par 0 ou 5.
  • • Les multiples de 10 se terminent toujours par 0.
  • • Les multiples de 3 sont des nombres dont la somme des chiffres est un multiple de 3.
Exemple pour les multiples de 3 : 375 → 3 + 7 + 5 = 15 (15 = 3 × 5), donc 375 est un multiple de 3.

Exercice 1 – Reconnaître les multiples

a Comment reconnaît-on les multiples de 2 ? de 3 ? de 5 ?
b Parmi ces nombres, lesquels sont des multiples de 2 : 123 ; 456 ; 789 ; 230 ; 54 ; 76 ; 75
c Parmi ces nombres, lesquels sont des multiples de 10 : 123 ; 456 ; 780 ; 230 ; 540 ; 76 ; 75
✅ Voir la correction ▼
a) Les multiples de 2 se terminent par 0, 2, 4, 6 ou 8. Les multiples de 3 ont une somme des chiffres qui est un multiple de 3. Les multiples de 5 se terminent par 0 ou 5.
b) Multiples de 2 : 456, 230, 54, 76 (ils se terminent par un chiffre pair)
c) Multiples de 10 : 780, 230, 540 (ils se terminent par 0)
🔎

Exercice 2 – Trouver des diviseurs

a Cite 5 diviseurs de 24.
b Cite tous les diviseurs de 12.
c Cite tous les diviseurs de 36.
✅ Voir la correction ▼
a) 5 diviseurs de 24 : 1, 2, 3, 4, 6 (on peut aussi citer 8, 12, 24)
b) Diviseurs de 12 : 1, 2, 3, 4, 6, 12
c) Diviseurs de 36 : 1, 2, 3, 4, 6, 9, 12, 18, 36
🔢

Exercice 3 – Citer des multiples

a Cite 3 multiples de 4.
b Cite 3 multiples de 7.
c Cite 3 multiples de 9.
d Parmi ces nombres, lesquels sont des multiples de 3 : 45 ; 82 ; 123 ; 256 ; 333 ; 710
e Parmi ces nombres, lesquels sont des multiples de 5 : 35 ; 72 ; 100 ; 213 ; 450 ; 88
✅ Voir la correction ▼
a) 3 multiples de 4 : 4, 8, 12 (ou 16, 20, 24…)
b) 3 multiples de 7 : 7, 14, 21 (ou 28, 35, 42…)
c) 3 multiples de 9 : 9, 18, 27 (ou 36, 45, 54…)
d) Multiples de 3 : 45 (4+5=9), 123 (1+2+3=6), 333 (3+3+3=9). Pas 82 (8+2=10), ni 256 (2+5+6=13), ni 710 (7+1+0=8).
e) Multiples de 5 : 35, 100, 450 (ils se terminent par 0 ou 5)
